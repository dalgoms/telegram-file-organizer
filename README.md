# Telegram File Organizer Bot

텔레그램에서 폴더 경로를 보내면 AI가 파일을 분석하여 자동으로 정리해주는 봇.

## 핵심 기능

- **AI 스마트 분류** — GPT가 파일명, 확장자, 크기, 날짜를 분석하여 최적의 폴더 구조 제안
- **날짜별 폴더** — 정리 시 `정리_2026-03-19/` 상위 폴더가 자동 생성되어 언제 정리했는지 한눈에 확인
- **미리보기 → 확인 → 실행** — 정리안을 먼저 보여주고, 확인 후 실행
- **정리 / 삭제 / 그만두기** — 미리보기 후 3가지 선택지 제공
- **Undo** — 언제든 마지막 정리를 되돌릴 수 있음
- **커스텀 규칙** — 자주 쓰는 정리 패턴을 규칙으로 저장
- **Google Drive 지원** — 로컬 폴더와 구글 드라이브 모두 정리 가능

### 귀차니즘을 위한 분석 기능

- **중복 파일 탐지** (`/dup`) — 같은 파일이 여러 폴더에 흩어져 있는지 찾아 낭비 용량 표시
- **버전 체인 분석** (`/ver`) — "보고서_최종", "보고서_최종_수정", "보고서_진짜최종" 같은 파일을 그룹핑
- **용량 분석** (`/size`) — 용량 TOP 파일 + 확장자별 용량 통계
- **아카이브 제안** (`/old`) — 올해가 아닌 오래된 파일을 연도별로 정리 제안
- **파일 검색** (`/find`) — "보고서", "KOBA" 등 키워드로 파일 검색
- **종합 리포트** (`/report`) — 위 분석을 한 번에 실행하여 폴더 상태를 진단
- **결과에서 바로 액션** — 모든 분석 결과에서 정리하기 / 삭제하기 버튼 제공

### 편의 기능

- **경로 바로가기** (`/path`) — "다운로드", "바탕화면" 같은 별칭으로 빠르게 접근
- **예약 스캔** (`/schedule`) — 매주 월요일 9시 자동 스캔 알림 설정
- **정리 통계 + 업적** (`/stats`) — 정리 횟수, 연속 기록, 칭호 시스템

## 명령어

| 명령어 | 기능 |
|--------|------|
| `/scan 경로` | 폴더 스캔 + AI 분류 미리보기 |
| `/scan 경로 -r` | 하위 폴더 포함 스캔 |
| `/run` | 미리보기 확인 후 실제 정리 |
| `/undo` | 마지막 정리 되돌리기 |
| `/history` | 정리 이력 보기 |
| `/rule` | 커스텀 규칙 목록 |
| `/rule 패턴 폴더` | 규칙 추가 |
| `/delrule 패턴` | 규칙 삭제 |
| `/gdrive 폴더ID` | 구글 드라이브 폴더 정리 |
| `/dup 경로` | 중복 파일 탐지 |
| `/ver 경로` | 버전 체인 분석 (최종/수정/v1 등) |
| `/size 경로` | 용량 TOP + 확장자별 통계 |
| `/old 경로` | 연도별 아카이브 제안 |
| `/find 키워드` | 파일명 검색 |
| `/report 경로` | 종합 리포트 (위 분석 전부) |
| `/path` | 경로 바로가기 보기 |
| `/path 이름 경로` | 바로가기 등록 |
| `/schedule` | 예약 스캔 관리 |
| `/stats` | 정리 기록 + 업적 |

## 사용 예시

```
# 다운로드 폴더 정리
/scan D:\Downloads

# 바탕화면 정리 (하위 폴더 포함)
/scan C:\Users\사용자\Desktop -r

# 별칭으로 간편 스캔
/scan 다운로드

# 경로만 보내도 동작
D:\프로젝트\마케팅자료

# 커스텀 규칙: PPT는 항상 문서/프레젠테이션으로
/rule *.pptx 문서/프레젠테이션

# 구글 드라이브 폴더 정리
/gdrive 1abc2def3ghi4jkl
```

## AI 분류 + 날짜 폴더 예시

입력:
```
2026_Q1_마케팅보고서_v3.pptx
IMG_20260318_142355.jpg
회의록_팀블로_0312.docx
budget_2026.xlsx
screenshot_234.png
```

AI 분류 결과 (미리보기):
```
--- 'Downloads' 정리 미리보기 ---
정리_2026-03-19/

  /문서/보고서/
    2026_Q1_마케팅보고서_v3.pptx
  /문서/회의록/
    회의록_팀블로_0312.docx
  /문서/재무/
    budget_2026.xlsx
  /이미지/사진/
    IMG_20260318_142355.jpg
  /이미지/스크린샷/
    screenshot_234.png

총 5개 파일을 5개 폴더로 정리해요.
```

실행 후 실제 폴더 구조:
```
Downloads/
  └─ 정리_2026-03-19/
       ├─ 문서/보고서/
       ├─ 문서/회의록/
       ├─ 문서/재무/
       ├─ 이미지/사진/
       └─ 이미지/스크린샷/
```

## 설치

### 1. 의존성 설치

```bash
cd telegram-file-organizer
pip install -r requirements.txt
```

### 2. 환경 변수 설정

`.env.example`을 복사하여 `.env` 파일을 만들고 값을 채워주세요:

```
TELEGRAM_BOT_TOKEN=your_telegram_bot_token
OPENAI_API_KEY=your_openai_api_key
DEVICE_NAME=집PC
```

### 3. 텔레그램 봇 토큰 발급

1. Telegram에서 @BotFather 검색
2. `/newbot` 입력 → 봇 이름과 username 설정
3. 발급된 토큰을 `.env`에 입력

### 4. 실행

```bash
python bot.py
```

### 5. Google Drive 연동 (선택)

1. [Google Cloud Console](https://console.cloud.google.com/)에서 프로젝트 생성
2. Google Drive API 활성화
3. OAuth 2.0 클라이언트 ID 생성 → `credentials.json` 다운로드
4. 프로젝트 루트에 `credentials.json` 배치
5. 첫 `/gdrive` 실행 시 브라우저에서 인증

## 안전 설계 — "사용자가 요청해야만 실행"

이 봇은 **사용자가 명시적으로 버튼을 누르지 않는 한 아무것도 이동하거나 삭제하지 않습니다.**
자동 정리, 자동 삭제 기능은 의도적으로 배제했습니다.

### 정리(이동)의 안전장치
| 단계 | 보호 |
|------|------|
| 스캔 | 미리보기만 보여줌 — 파일에 손대지 않음 |
| 실행 | "정리하기" 버튼을 직접 눌러야 실행 |
| 복구 | `/undo` 한 번이면 원래대로 복원 |
| 만료 | 5분 경과 시 스캔 자동 만료 |

### 삭제의 안전장치
| 단계 | 보호 |
|------|------|
| 1차 | "삭제하기" 버튼 클릭 → 파일 목록 + 용량 확인 |
| 2차 | "정말 N개 삭제할게요" 재확인 → 이때만 실행 |
| 취소 | 어디서든 "취소"로 중단 가능 |

### 시스템 보호
- `C:\Windows`, `C:\Program Files` 등 시스템 폴더 자동 차단
- GPT 반환 폴더명에서 `../` 경로 탈출 패턴 자동 제거
- 심볼릭 링크 자동 건너뜀
- 사용 중인 파일(잠금) 감지 후 건너뜀
- Windows 260자 경로 제한 감지
- 동일 파일명 충돌 시 자동 번호 부여
- 매 파일 이동마다 이력 즉시 저장 (원자적 쓰기)
- 시스템 파일(`.git`, `node_modules`, `.env` 등) 자동 스킵

## 기술 스택

- Python 3.12+
- python-telegram-bot[job-queue] — 텔레그램 봇 인터페이스 + 예약 스캔
- OpenAI API (gpt-4o-mini) — AI 파일 분류
- Google Drive API — 클라우드 파일 관리
- pathlib / shutil — 로컬 파일 조작
- hashlib MD5 — 중복 파일 탐지
